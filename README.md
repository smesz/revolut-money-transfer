# Money Transfer REST API
REST API for money transfers between accounts.

## Technologies used
* Java 8
* Dropwizard
* Liquibase
* H2 in-memory DB
* JPA 2 / Hibernate
* Spock Framework

## API documentation
Application exposes following REST endpoints

| HTTP method  | Endpoint | Request  | Description |
|---|---|---|---|
| POST  | /account | ''''''      | Creates new account in the system
